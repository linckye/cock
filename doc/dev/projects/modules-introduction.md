## Modules introduction

模块划分如下：

- client：客户端包
- common：通用库
- doc：文档
  - dev：开发者文档
- function-api：function 对外接口
  - java：java function 接口
- repositories：存储包
- scheduler：scheduler 模块
  - client：scheduler 客户端包
  - core：scheduler 核心包
  - deployer：scheduler 发布包
  - repositories：scheduler 存储包
  - services：scheduler 服务包
- services：服务包
- test：测试包
- web：web 模块
  - client：scheduler 客户端包
  - controllers：scheduler 控制器包
  - deployer：web 发布包
  - repositories：web 存储包
  - services：web 服务包
- worker：worker 模块
  - client：客户端包
  - core：核心包
  - deployer：发布包
  - repositories：存储包
  - services：服务包
